trust

Function

The trust command specifies the priority to be mapped for packets.

The undo trust command cancels the configuration.

By default, packets are processed based on the mapping of the outer 802.1p priority.

Format

For CE6860-SAN, CE6866K, CE6866, CE6860-HAM, CE8850-SAN, CE8851K, CE8851-32CQ8DQ-P, CE8850-HAM, CE6855-48XS8CQ, CE6885, CE6885-T, CE6885-LL (standard forwarding mode), CE6863E-48S8CQ, CE6885-SAN, CE8855, CE8851-32CQ4BQ, CE6820H, CE6820H-K, CE6820S, CE6863H, CE6863H-K, CE6881H, CE6881H-K:

trust { 8021p { inner | outer } | dscp }

For CE6860-SAN, CE6866K, CE6866, CE6860-HAM, CE8850-SAN, CE8851K, CE8851-32CQ8DQ-P, CE8850-HAM, CE6855-48XS8CQ, CE6885, CE6885-T, CE6885-LL (standard forwarding mode), CE6863E-48S8CQ, CE6885-SAN, CE8855, CE8851-32CQ4BQ, CE6820H, CE6820H-K, CE6820S, CE6863H, CE6863H-K, CE6881H, CE6881H-K, CE6885-LL (low latency mode):

undo trust

For CE6885-LL (low latency mode):

trust { 8021p outer | dscp }

Usage Guidelines

Usage Scenario

A DiffServ domain defines the mapping between PHBs/colors and two types of priorities, namely, 802.1p priority and DSCP priority. When the DiffServ domain is bound to an interface, you can run the trust command to specify the priority to be mapped.

To set the same priority to be trusted on multiple interfaces, you can perform the configuration on a port group to reduce the workload.

Precautions

  • After an Ethernet interface is switched to Layer 3 mode using the undo portswitch command, the interface trusts DSCP priorities by default, and the priority to be mapped for packets cannot be specified on the interface.
  • If the trust 8021p { inner | outer } command is configured:
  • For untagged packets, the trusted priority is the interface priority.
  • For single-tagged packets, the trusted priority is the priority of the tag.
  • If you run the trust command multiple times in the same interface view, only the latest configuration takes effect.
  • Layer 3 sub-interfaces do not support the trust 8021p inner command.

Example

# Configure 100GE1/0/1 to trust DSCP priorities.
<HUAWEI> system-view
[~HUAWEI] interface 100GE 1/0/1
[~HUAWEI-100GE1/0/1] trust dscp
