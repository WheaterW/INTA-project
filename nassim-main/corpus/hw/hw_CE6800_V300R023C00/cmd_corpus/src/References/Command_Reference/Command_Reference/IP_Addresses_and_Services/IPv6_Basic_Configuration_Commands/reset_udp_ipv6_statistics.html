reset udp ipv6 statistics

Function

The reset udp ipv6 statistics command clears IPv6 UDP packet statistics.

This command is supported only on the CE6863H, CE6863H-K, CE6860-SAN, CE6866K, CE6866, CE6860-HAM, CE6855-48XS8CQ, CE6885-SAN, CE8850-SAN, CE8855, CE8851-32CQ4BQ, CE8851K, CE8851-32CQ8DQ-P, CE8850-HAM, CE6881H, CE6881H-K, CE6820H, CE6820H-K, CE6820S, CE6885, CE6885-T, CE6885-LL (standard forwarding mode) and CE6863E-48S8CQ.

Format

reset udp ipv6 statistics

Parameters

None

Views

User view

Default Level

2: Configuration level

Usage Guidelines

Usage Scenario

Before collecting the IPv6 UDP packet statistics in a specified period, you need to run the reset udp ipv6 statistics command to clear previous IPv6 UDP packet statistics. After that, run the display udp ipv6 statistics command to view the IPv6 UDP packet statistics.

Configuration Impact

The reset udp ipv6 statistics command is used to clear IPv6 UDP packet statistics. Therefore, confirm the action before running this command.

Example

# Clear IPv6 UDP packet statistics.
<HUAWEI> reset udp ipv6 statistics
