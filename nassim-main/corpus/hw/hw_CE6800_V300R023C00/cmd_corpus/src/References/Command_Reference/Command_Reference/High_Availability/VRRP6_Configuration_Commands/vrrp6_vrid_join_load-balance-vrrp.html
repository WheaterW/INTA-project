vrrp6 vrid join load-balance-vrrp

Function

The vrrp6 vrid join load-balance-vrrp command adds a specified VRRP6 backup group to a Virtual Router Redundancy Protocol for IPv6 (VRRP6) load-balance redundancy group (LBRG).

The undo vrrp6 vrid join load-balance-vrrp command deletes a specified VRRP6 backup group from a VRRP6 LBRG.

By default, no VRRP6 backup group is added to a VRRP6 LBRG.

This command is supported only on the CE6863H, CE6863H-K, CE6860-SAN, CE6866K, CE6866, CE6860-HAM, CE6855-48XS8CQ, CE6885-SAN, CE8850-SAN, CE8855, CE8851-32CQ4BQ, CE8851K, CE8851-32CQ8DQ-P, CE8850-HAM, CE6881H, CE6881H-K, CE6820H, CE6820H-K, CE6820S, CE6885, CE6885-T, CE6885-LL (standard forwarding mode) and CE6863E-48S8CQ.

Format

vrrp6 vrid virtual-router-id join load-balance-vrrp vrid lb-vrid-value

undo vrrp6 vrid virtual-router-id join load-balance-vrrp

Parameters

Parameter Description Value
vrid virtual-router-id

Specifies the ID of a VRRP6 backup group.

The value is an integer ranging from 1 to 255.
vrid lb-vrid-value

Specifies the ID of a VRRP6 LBRG.

The value is an integer ranging from 1 to 255.

Default Level

Usage Guidelines

Usage Scenario

To implement VRRP6 single-gateway load balancing, run the vrrp6 vrid join load-balance-vrrp command to add a specified VRRP6 backup group to a VRRP6 LBRG. The VRRP6 LBRG uses the virtual MAC addresses of the VRRP6 backup groups in it to respond to different users' NS request packets destined for virtual IPv6 addresses. The master device of a specified VRRP6 backup group forwards the traffic of a specified user, implementing load balancing.

Prerequisites

A VRRP6 LBRG has been created using the vrrp6 vrid vrid-value load-balance command.

  • If a VRRP6 group is used as a VRRP6 LBRG, you must configure a virtual IPv6 address to the VRRP6 group.
  • If a VRRP6 group is used as a member group of a VRRP6 LBRG, you do not need to configure a virtual IPv6 address for the VRRP6 group.

Example

# Create a VRRP6 LBRG and add a VRRP6 backup group to the VRRP6 LBRG.
<HUAWEI> system-view
[~HUAWEI] interface 100GE1/0/1
[~HUAWEI-100GE1/0/1] undo portswitch
[*HUAWEI-100GE1/0/1] ipv6 enable
[*HUAWEI-100GE1/0/1] ipv6 address 2001:db8::1 64
[*HUAWEI-100GE1/0/1] vrrp6 vrid 1 virtual-ip FE80::7 link-local
[*HUAWEI-100GE1/0/1] vrrp6 vrid 1 virtual-ip 2001:db8::2
[*HUAWEI-100GE1/0/1] vrrp6 vrid 1 load-balance
[*HUAWEI-100GE1/0/1] vrrp6 vrid 2
[*HUAWEI-100GE1/0/1] vrrp6 vrid 2 join load-balance-vrrp vrid 1
