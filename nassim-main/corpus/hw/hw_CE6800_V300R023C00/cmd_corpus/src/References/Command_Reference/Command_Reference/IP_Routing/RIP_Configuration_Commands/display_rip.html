display rip

Function

The display rip command displays the operating status and configurations of a RIP process.

The display ripng command displays the current operating status and configurations of a RIPng process.

The display default-parameter rip command displays the default values of all the configurable parameters of RIP.

The display default-parameter ripng command displays default RIPng configurations.

Format

display rip [ process-id | vpn-instance vpn-instance-name ]

display default-parameter rip

display ripng [ process-id | vpn-instance vpn-instance-name ]

display default-parameter ripng

Parameters

Parameter Description Value
process-id

Specifies the ID of a process.

The value is an integer ranging from 1 to 4294967295.
vpn-instance vpn-instance-name

Specifies the name of a VPN instance.

The value is a string of 1 to 31 case-sensitive characters. It cannot contain spaces. The VPN instance name cannot be _public_. If the character string is quoted by double quotation marks, the character string can contain spaces.

Views

All views

Default Level

1: Monitoring level

Usage Guidelines

Usage Scenario

To check the running status and configurations of RIP processes on a local device, run the display rip command.

After basic RIPng functions are configured, you can run this command to view the configured information.

You can run the display default-parameter rip command either in a process or on an interface.

Even if the default RIPng configurations have been changed, this command displays the original default RIPng configurations.

You can run the display default-parameter ripng command either in a process or on an interface.

Precautions

If no process ID is specified, running the display ripng command displays the parameters of all configured RIPng processes.
