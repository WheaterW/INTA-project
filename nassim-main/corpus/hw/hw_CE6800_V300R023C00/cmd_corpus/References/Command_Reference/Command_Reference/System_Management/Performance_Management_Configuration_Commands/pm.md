pm
==

pm

Function
--------



The **pm** command displays the PM view.




Format
------

**pm**


Parameters
----------

None

Views
-----

System view


Default Level
-------------

2: Configuration level


Usage Guidelines
----------------

**Usage Scenario**

To enable the performance statistics function of PM, run the pm command to display the PM view.


Example
-------

# Enter the PM view.
```
<HUAWEI> system-view
[~HUAWEI] pm
[~HUAWEI-pm]

```
