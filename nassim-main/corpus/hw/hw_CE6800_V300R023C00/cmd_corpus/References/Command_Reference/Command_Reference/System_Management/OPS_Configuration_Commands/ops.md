ops
===

ops

Function
--------



The **ops** command displays the OPS view.



By default, the OPS view exists.


Format
------

**ops**


Parameters
----------

None

Views
-----

System view


Default Level
-------------

3: Management level


Usage Guidelines
----------------

**Usage Scenario**

To enter the OPS view; run the **ops** command.

**Precautions**


Example
-------

# Display the OPS view.
```
<HUAWEI> system-view
[~HUAWEI] ops
[~HUAWEI-ops]

```
