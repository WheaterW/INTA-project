oas
===

oas

Function
--------



The **oas** command is used to enter the oas view.



By default, the OAS view cannot be displayed.


Format
------

**oas**


Parameters
----------

None

Views
-----

System view


Default Level
-------------

3: Management level


Usage Guidelines
----------------

After OAS is enabled, you can perform OAS-related operations only in the OAS view.


Example
-------

# Enter the OAS view.
```
<HUAWEI> system-view
[~HUAWEI] oas
[~HUAWEI-oas]

```
