multicast ipv6 verification pim routing-table
=============================================

multicast ipv6 verification pim routing-table

Function
--------



The **multicast ipv6 verification pim routing-table** command enables multicast entry verification and manually restores the restricted multicast routing entries.



![](../public_sys-resources/note_3.0-en-us.png) 

This command is supported only on the CE6863H, CE6863H-K, CE6860-SAN, CE6866K, CE6866, CE6860-HAM, CE6855-48XS8CQ, CE6885-SAN, CE8850-SAN, CE8855, CE8851-32CQ4BQ, CE8851K, CE8851-32CQ8DQ-P, CE8850-HAM, CE6881H, CE6881H-K, CE6820H, CE6820H-K, CE6820S, CE6885, CE6885-T, CE6885-LL (standard forwarding mode) and CE6863E-48S8CQ.



Format
------

**multicast ipv6 verification pim routing-table**


Parameters
----------

None

Views
-----

User view


Default Level
-------------

3: Management level


Usage Guidelines
----------------

To enable multicast ipv6 entry verification, run the multicast ipv6 verification pim routing-table command. Restricted multicast routing entries can be manually restored.


Example
-------

# Enable multicast ipv6 entry verification,restricted multicast ipv6 routing entries can be manually restored.
```
<HUAWEI> multicast ipv6 verification pim routing-table

```
