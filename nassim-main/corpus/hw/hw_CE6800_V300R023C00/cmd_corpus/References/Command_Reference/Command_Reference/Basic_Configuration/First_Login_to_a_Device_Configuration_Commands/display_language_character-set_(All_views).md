display language character-set (All views)
==========================================

display language character-set (All views)

Function
--------



The **display language character-set** command displays the character set type of the current device.




Format
------

**display language character-set**


Parameters
----------

None

Views
-----

All views


Default Level
-------------

0: Visit level


Usage Guidelines
----------------

You can run this command to view the character set type of the current device.


Example
-------

![](../public_sys-resources/note_3.0-en-us.png) 

The actual command output varies according to the device. The command output here is only an example.


# Display character set information configured in the system.
```
<HUAWEI> display language character-set
Current language character set encode : UTF-8

```

**Table 1** Description of the **display language character-set (All views)** command output
| Item | Description |
| --- | --- |
| Current language character set encode | Current character set type. |
