grpc
====

grpc

Function
--------



The **grpc** command displays the gRPC view.




Format
------

**grpc**


Parameters
----------

None

Views
-----

System view


Default Level
-------------

3: Management level


Usage Guidelines
----------------

**Usage Scenario**

The grpc command is used to display the gRPC view. After the gRPC view is displayed, you can run related commands in the view.


Example
-------

# Display the gRPC view.
```
<HUAWEI> system-view
[~HUAWEI] grpc

```
