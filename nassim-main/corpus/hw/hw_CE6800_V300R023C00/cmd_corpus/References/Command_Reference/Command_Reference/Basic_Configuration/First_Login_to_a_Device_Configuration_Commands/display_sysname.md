display sysname
===============

display sysname

Function
--------



The **display sysname** command displays a device host name.




Format
------

**display sysname**


Parameters
----------

None

Views
-----

All views


Default Level
-------------

3: Management level


Usage Guidelines
----------------

You can run this command to view the host name of the current device.


Example
-------

![](../public_sys-resources/note_3.0-en-us.png)
