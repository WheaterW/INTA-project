display python version
======================

display python version

Function
--------



The **display python version** command displays information about the Python version that is in use.




Format
------

**display python version**


Parameters
----------

None

Views
-----

All views


Default Level
-------------

3: Management level


Usage Guidelines
----------------

**Usage Scenario**

To check information about the Python version that is in use, run the display python version command. The command output helps determine whether the Python version meets requirements.


Example
-------

![](../public_sys-resources/note_3.0-en-us.png) 

The actual command output varies according to the device. The command output here is only an example.


# Display information about the Python version that is in use.

<HUAWEI> display python version
