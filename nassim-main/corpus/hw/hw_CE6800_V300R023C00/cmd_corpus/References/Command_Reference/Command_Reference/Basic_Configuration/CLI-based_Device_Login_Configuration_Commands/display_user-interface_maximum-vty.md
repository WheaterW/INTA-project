display user-interface maximum-vty
==================================

display user-interface maximum-vty

Function
--------



The **display user-interface maximum-vty** command displays the configured maximum VTY value.




Format
------

**display user-interface maximum-vty**


Parameters
----------

None

Views
-----

All views


Default Level
-------------

3: Management level


Usage Guidelines
----------------

**Usage Scenario**

You can run this command to view the maximum number of VTY users.


Example
-------

![](../public_sys-resources/note_3.0-en-us.png) 

The actual command output varies according to the device. The command output here is only an example.


# Display maximum number of VTY user interfaces.
```
<HUAWEI> display user-interface maximum-vty
Maximum of VTY user:5

```

**Table 1** Description of the **display user-interface maximum-vty** command output
| Item | Description |
| --- | --- |
| Maximum of VTY user | Maximum number of login users through VTY. |
