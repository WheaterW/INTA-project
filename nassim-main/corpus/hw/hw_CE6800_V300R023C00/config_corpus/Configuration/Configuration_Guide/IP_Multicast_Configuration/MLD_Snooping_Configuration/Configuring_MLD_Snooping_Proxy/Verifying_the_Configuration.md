Verifying the Configuration
===========================

Verifying the Configuration

#### Procedure

* Run the [**display mld snooping**](cmdqueryname=display+mld+snooping) [ **vlan** [ *vlan-id* ] ] **configuration** command to check the MLD snooping proxy configuration of a VLAN.
* Run the [**display mld snooping querier**](cmdqueryname=display+mld+snooping+querier) **vlan** [ *vlan-id* ] command to check the MLD snooping querier state.
