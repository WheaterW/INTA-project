Verifying the Configuration
===========================

Verifying the Configuration

#### Procedure

* Run the [**display ospfv3**](cmdqueryname=display+ospfv3) [ *process-id* ] command to check the authentication mode configured for the OSPFv3 process based on the **Authentication** field in the command output.
* Run the [**display ospfv3**](cmdqueryname=display+ospfv3) [ *process-id*] **area** [ *area-id* ] command to check the authentication mode configured for the OSPFv3 area based on the **Authentication** field in the command output.
* Run the [**display ospfv3**](cmdqueryname=display+ospfv3) [ *process-id* ] **interface** [ **area** *area-id* ] [ *interface-type* *interface-number* ] command to check the authentication mode configured for the OSPFv3 interface based on the **Authentication** field in the command output.
