| DHCPv4 Snooping | When DHCPv4 snooping is configured in a VXLAN scenario, users need to access the network through VLANs on the access side, but cannot access the network through Layer 2 sub-interfaces or tunnels on the access side. In this scenario, DHCP snooping does not verify packets when forwarding packets based on the service logic.
