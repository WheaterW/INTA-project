For the CE6800 Series:

1. When the MAC address of a Layer 3 interface is a virtual MAC address, the SMAC address of the packets forwarded from the interface is not the virtual MAC address but the system MAC address.

2. If the device has only one system MAC address, the first system MAC address is used. If the device has two system MAC addresses, the second system MAC address is used. If the device has three or more system MAC addresses, the third system MAC address is used.
