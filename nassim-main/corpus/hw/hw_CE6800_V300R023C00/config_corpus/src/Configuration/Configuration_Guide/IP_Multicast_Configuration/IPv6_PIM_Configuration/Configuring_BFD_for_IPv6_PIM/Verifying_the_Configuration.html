Verifying the Configuration

Procedure

  • Run the display pim ipv6 bfd session statistics command to check global BFD statistics.
  • Run the display pim ipv6 bfd session [ neighbor ipv6-link-local-address | interface { interface-name | interface-type interface-number } ] * command to check BFD statistics on an interface.
