Verifying the Configuration

Procedure

  • Run the display diffserv domain [ brief | ds-domain-name ] command to check the DiffServ domain configuration.
  • Run the display qos queue statistics { interface interface-name | interface interface-type interface-number | slot slotid } command to check queue-based traffic statistics.
  • Run the display qos configuration interface [ { interface-type interface-number | interface-name } ] command to check all QoS configurations on an interface.
