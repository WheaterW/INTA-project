Verifying the Configuration

Procedure

  1. Run the display dpfr cache [ ip-address ] slot slot-id [ chip chip-id ] command to view detailed information about the fault table for DPFR.
  2. Run the display dpfr cache statistics slot slot-id [ chip chip-id ] command to view the fault table statistics about DPFR.
  3. Run the display dpfr event-log [ recover | resource-fail | port-status ] slot slot-id command to view the event logs of DPFR.
