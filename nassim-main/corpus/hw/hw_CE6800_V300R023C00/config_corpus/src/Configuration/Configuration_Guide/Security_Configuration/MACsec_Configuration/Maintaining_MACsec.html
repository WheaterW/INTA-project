Maintaining MACsec

Procedure

  1. Run the reset mka statistics interface { interface-name | interface-type interface-number } command in the user view to clear MKA protocol packet statistics on a specified interface.
  2. Run the reset macsec statistics interface { interface-name | interface-type interface-number } command in the user view to clear statistics about data packets protected by MACsec on a specified interface.
