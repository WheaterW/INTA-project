Overview of IPv6 L3VPN

Definition

VPN technology is used to establish private networks over public networks.

Depending on whether CEs reside on an IPv4 or IPv6 network, L3VPN can be classified as IPv4 L3VPN or IPv6 L3VPN. For more information about IPv6 L3VPN, see Overview of IPv4 L3VPN.
