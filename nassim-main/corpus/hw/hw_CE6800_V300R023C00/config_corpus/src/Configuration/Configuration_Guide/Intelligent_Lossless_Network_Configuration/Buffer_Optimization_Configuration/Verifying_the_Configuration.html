Verifying the Configuration

Procedure

  • For the CE6866, CE6860-SAN, CE6866K, CE6860-HAM, CE8851-32CQ8DQ-P, CE8850-SAN, CE8851K, CE8850-HAM: Run the display buffer optimization configuration slot slot-id command to check buffer optimization information, including the buffer size of each plane, buffer configuration of each interface, and calculation result of the distance-based headroom buffer check.
  • For the CE6855-48XS8CQ, CE6863E-48S8CQ, CE6885, CE6885-LL (standard forwarding mode), CE6885-SAN, CE6885-T, CE8851-32CQ4BQ, CE8855: Run the display long-distance configuration slot slot-id command to check the long-distance configuration of each interface and the calculation result of the distance-based headroom buffer check.
