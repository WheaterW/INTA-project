Verifying the Configuration

Procedure

  • Run the display igmp snooping explicit-tracking { vlan [ vlan-id [ interface { interface-type interface-number | interface-name } ] ] } [ [ source-address source-address ] group-address group-address ] [ [ host-address host-address ] verbose ] command to check information about tracked hosts in a VLAN.
  • Run the display igmp snooping [ vlan [ vlan-id ] ] configuration command to check the IGMP snooping configuration.

    By viewing the IGMP snooping configuration, you can check the IGMP snooping membership update configuration in a VLAN.
