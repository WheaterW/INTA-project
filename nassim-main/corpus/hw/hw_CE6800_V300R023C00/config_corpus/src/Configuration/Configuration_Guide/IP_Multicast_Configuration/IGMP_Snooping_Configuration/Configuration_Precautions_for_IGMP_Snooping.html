Configuration Precautions for IGMP Snooping

Licensing Requirements

IGMP snooping is a basic function included in the basic software function license. The basic software function license has been installed and activated before delivery. You do not need to manually activate it.

IGMPv2 Join messages are reported as traffic. The protocol considers that the source is alive and adds the ACT flag to multicast routing entries.

IGMPv2 Leave messages are also reported as traffic. After receiving an IGMPv2 Leave message, the device deletes only the multicast outbound interface entry but not the multicast group entry. The multicast group entry can be deleted only after the default multicast aging time (210s, which is configurable) expires.
