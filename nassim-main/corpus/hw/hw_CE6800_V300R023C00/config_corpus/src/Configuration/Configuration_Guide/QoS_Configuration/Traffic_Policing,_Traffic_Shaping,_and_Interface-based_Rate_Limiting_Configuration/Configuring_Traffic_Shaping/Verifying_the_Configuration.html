Verifying the Configuration

Procedure

  • Run the display qos configuration interface [ { interface-type interface-number | interface-name } ] command to check all QoS configurations on an interface.
  • Run the display qos queue statistics { slot slotid | interface { interface-type interface-number | interface-name } } command to check queue-based traffic statistics.
