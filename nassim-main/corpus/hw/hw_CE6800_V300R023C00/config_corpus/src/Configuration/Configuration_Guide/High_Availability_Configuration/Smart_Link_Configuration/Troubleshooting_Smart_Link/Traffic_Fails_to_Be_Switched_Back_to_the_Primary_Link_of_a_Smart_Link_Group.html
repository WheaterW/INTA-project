Traffic Fails to Be Switched Back to the Primary Link of a Smart Link Group

Possible Causes

The switchback function may not be enabled for the Smart Link group.

Procedure

  • Run the display smart link group { all | group-id } command to check the state of the Smart Link group and whether the switchback function is enabled.
