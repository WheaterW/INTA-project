Verifying the Configuration

Procedure

  • Run the display bgp ipv6 peer { ipv6-address | ipv4-address } verbose command to check detailed BGP4+ peer information.
  • Run the display bgp ipv6 peer ipv6-address log-info command to check the log information of BGP4+ peers.
  • Run the display bgp ipv6 routing-table command to check information about BGP4+ routes.
  • Run the display bgp ipv6 routing-table [ ipv6-address prefix-length ] command to check BGP4+ routing information.
