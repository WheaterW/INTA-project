Verifying the Configuration

Procedure

  1. Run the display ip statistics [ interface interface-type interface-number ] command to check IP traffic statistics.
  2. Run the display forward mode command to check the configured and currently used packet forwarding modes.

    The CE6885-LL in low-latency mode does not support the packet forwarding mode.
