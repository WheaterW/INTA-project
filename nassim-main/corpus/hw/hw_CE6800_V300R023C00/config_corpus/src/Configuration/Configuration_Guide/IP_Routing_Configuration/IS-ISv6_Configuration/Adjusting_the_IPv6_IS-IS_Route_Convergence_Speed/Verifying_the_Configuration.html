Verifying the Configuration

Procedure

  • Run the display isis interface [ verbose ] [ vpn-instance vpn-instance-name ] command to check information about IS-IS packets sent by IPv6 IS-IS interfaces.
  • Run the display isis route [ process-id | vpn-instance vpn-instance-name ] ipv6 [ verbose | [ level-1 | level-2 ] | ipv6-address [ prefix-length ] ] * command to check preferences of IPv6 IS-IS routes.
