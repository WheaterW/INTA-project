Stopping a Maintenance Assistant

Context

If a maintenance assistant is no longer needed, you can stop it.

Stopping a running maintenance assistant interrupts the task of the assistant. Exercise caution when you perform this operation.

Procedure

  1. Enter the system view.

    system-view

  2. Enter the OPS view.

    ops

  3. Stop a script assistant.

    shutdown script-assistant script-name

    To restart the script assistant, run the undo shutdown script-assistant script-name command.

  4. Commit the configuration.

    commit
