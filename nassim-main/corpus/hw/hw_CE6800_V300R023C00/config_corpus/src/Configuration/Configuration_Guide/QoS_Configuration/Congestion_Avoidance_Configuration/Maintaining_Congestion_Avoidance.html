Maintaining Congestion Avoidance

Context

Statistics cannot be restored after they are cleared. Exercise caution when clearing the statistics.

Procedure

  • Clear queue-based traffic statistics.

    reset qos queue statistics { interface { interface-type interface-number | interface-name } | slot slot-id }

  • Clear statistics on packets with the ECN flag on an interface.

    reset qos ecn statistics [ interface { interface-type interface-number | interface-name } ]

    Statistics on packets with the ECN flag on an interface can be cleared only on the CE6866, CE6860-SAN, CE6866K, CE6860-HAM, CE8851-32CQ8DQ-P, CE8850-SAN, CE8851K, CE8850-HAM, CE8855, CE8851-32CQ4BQ, CE6855-48XS8CQ, CE6885, CE6885-SAN, CE6885-LL, CE6885-T, CE6863E-48S8CQ.
