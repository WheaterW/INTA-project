Definition and Classification of Load Balancing

Load balancing allows a network node to distribute traffic among multiple links for forwarding. It is classified as route, tunnel, or trunk load balancing.

Route Load Balancing

Route load balancing enables traffic to be balanced among multiple forwarding paths to the same destination. If there are multiple routes with the same destination address and mask but different next hops, outbound interfaces, or tunnel IDs, route load balancing can be implemented, as shown in Figure 1.

Tunnel Load Balancing
