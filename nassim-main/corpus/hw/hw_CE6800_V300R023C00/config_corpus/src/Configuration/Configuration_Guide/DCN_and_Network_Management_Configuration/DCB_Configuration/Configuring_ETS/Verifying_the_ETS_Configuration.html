Verifying the ETS Configuration

Prerequisites

All the ETS configurations are complete.

Procedure

  • Run the display dcb ets-profile [ profile-name ] command to check the ETS profile configuration.
  • Run the display dcb [ interface { interface-name | interface-type interface-num } ] command to check the DCB configuration and negotiation status.
