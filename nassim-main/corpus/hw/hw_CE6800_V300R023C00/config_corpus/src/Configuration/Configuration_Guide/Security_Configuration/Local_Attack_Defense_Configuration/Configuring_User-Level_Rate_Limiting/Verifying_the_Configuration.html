Verifying the Configuration

Procedure

  • Run the display cpu-defend host-car [ mac-address mac-address ] statistics [ slot slot-id ] command to check the number of packets discarded due to user-level rate limiting.
  • Run the display cpu-defend host-car drop-packet record [ car-id car-id ] [ slot slot-id ] command to check records of packet loss caused by user-level rate limiting.
