Verifying the Configuration

Procedure

  1. Run the display dhcp snooping command to check DHCP snooping running information.
  2. Run the display dhcp snooping configuration command to check the DHCP snooping configuration.
  3. Run the display user-bind static command to check information about the configured static binding table, including the IP address, MAC address, VLAN, and interface information. Check the information about the DHCP snooping binding table.
  4. Run the display dhcp snooping statistics command to check statistics about various DHCP messages received by the device.
