Verifying the Configuration

Procedure

  • Run the display ipv6 nd source-mac [ interface { interface-name | interface-type interface-num } ] command to check information about ND attack entries with fixed source MAC addresses.
  • Run the display ipv6 nd source-mac statistics [ interface { interface-name | interface-type interface-num } ] command to check statistics about dropped ND attack messages with fixed source MAC addresses.
