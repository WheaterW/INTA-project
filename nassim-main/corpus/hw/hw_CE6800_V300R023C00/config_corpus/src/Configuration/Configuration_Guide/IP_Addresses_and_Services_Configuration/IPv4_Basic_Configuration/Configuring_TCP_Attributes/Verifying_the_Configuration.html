Verifying the Configuration

Procedure

  1. Run the display tcp status [ local-ip ipv4-address ] [ local-port local-port-number ] [ remote-ip ipv4-address ] [ remote-port remote-port-number ] [ cid cid ] [ socket-id socket-id ] command to check the TCP connection status.
  2. Run the display tcp statistics command to check TCP traffic statistics.
