Verifying the Configuration

Procedure

  • Run the display pim [ vpn-instance vpn-instance-name | all-instance ] bsr-info command to check information about the BSR in a PIM-SM domain.
  • Run the display pim [ vpn-instance vpn-instance-name | all-instance ] rp-info [ group-address ] command to check information about the RPs in a PIM-SM domain.
