Checking Integrated Route Statistics of IPv4 VPN Instances

Procedure

  • Run the display ip routing-table { vpn-instance vpn-instance-name | all-vpn-instance } statistics command to check integrated route statistics of a specified IPv4 VPN instance or all IPv4 VPN instances.
