After the OAS function is enabled, the container daemon process occupies 60 MB memory, and the container instance with no running service occupies 10 MB memory. Therefore, it is recommended that the total reserved physical memory be less than or equal to 30% of the total physical memory. The formula is as follows: Total reserved physical memory = 60 + (Memory required by third-party containers + 10) x Number of containers.
