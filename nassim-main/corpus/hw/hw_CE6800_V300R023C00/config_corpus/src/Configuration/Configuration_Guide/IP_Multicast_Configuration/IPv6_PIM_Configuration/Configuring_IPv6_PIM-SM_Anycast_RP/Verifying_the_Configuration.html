Verifying the Configuration

Procedure

  • Run the display pim ipv6 [ vpn-instance vpn-instance-name | all-instance ] rp-info [ ipv6-group-address ] command to check RP information.
  • Run the display pim ipv6 [ vpn-instance vpn-instance-name | all-instance ] routing-table command to check the IPv6 PIM routing table.
