Maintaining an Image

Procedure

The OAS allows users to upgrade and uninstall the image software package. For details, see Table 1.

Table 1 Image maintenance operations

Operation

Command

Description

Upgrade the image software package.

upgrade application-software software-path

After the image software package is upgraded, the corresponding applications are also upgraded.

Uninstall the image software package.

uninstall application-software software-name

Before uninstalling the image software package, you need to delete related applications.
