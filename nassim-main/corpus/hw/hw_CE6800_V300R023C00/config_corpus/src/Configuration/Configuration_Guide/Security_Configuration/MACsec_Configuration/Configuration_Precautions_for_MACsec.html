MACsec is configured on an interface and its peer, and the interface can receive mirrored packets from the observing port. If the mirrored packets contain MKA protocol packets, the interface can receive both MKA protocol packets and mirrored MKA protocol packets from the peer, as a result, the port identifies two peers (MI is different). The device does not support multi-point connections. As a result, the protocol cannot be established, and data traffic cannot be transmitted on the observing port.
