Verifying the Configuration

Procedure

  • Run the display ospf [ process-id ] peer command to check OSPF neighbor information.
  • Run the display ospfv3 [ process-id ] peer command to check OSPFv3 neighbor information.
  • Run the display bgp peer command to check BGP peer information.
  • Run the display bgp ipv6 peer command to check BGP4+ peer information.
