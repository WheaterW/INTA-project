Packet loss visualization

For the CE6800 Series:

1. The maximum length of obtained packets discarded due to full buffer is 180 bytes, including the original packet length and frame header.

2. The packet content exceeding 180 bytes cannot be parsed, and the corresponding field in the flow table is filled with 0.

3. If the packet length exceeds 180 bytes, the original packet length cannot be counted in statistics about bytes in the flow table. Instead, the packet length is accumulated based on the truncated packets.
