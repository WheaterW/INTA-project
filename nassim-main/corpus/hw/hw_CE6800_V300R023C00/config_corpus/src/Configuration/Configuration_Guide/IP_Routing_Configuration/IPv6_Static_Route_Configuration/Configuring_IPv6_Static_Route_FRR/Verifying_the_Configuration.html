Verifying the Configuration

Procedure

  • Run the display ipv6 routing-table verbose command to check backup outbound interfaces and backup next hops in the routing table.
  • Run the display ipv6 routing-table ipv6-address [ prefix-length ] [ longer-match ] verbose command to check the backup outbound interface and backup next hop of the specified route in the routing table.
