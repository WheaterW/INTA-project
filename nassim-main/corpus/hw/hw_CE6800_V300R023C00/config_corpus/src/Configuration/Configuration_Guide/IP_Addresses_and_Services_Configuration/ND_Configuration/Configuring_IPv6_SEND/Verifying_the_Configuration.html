Verifying the Configuration

Procedure

  • Run the display ipv6 security interface { interface-name | interface-type interface-num } command to check the IPv6 SEND configuration.
  • Run the display ipv6 nd security timestamp { interface-name | interface-type interface-num } command to check the timestamp of an IPv6 SEND message.
  • Run the display ipv6 nd security nonce { interface-name | interface-type interface-num } command to check the Nonce value of an IPv6 SEND message.
  • Run the display ipv6 nd security statistics { interface-name | interface-type interface-num } command to check statistics on IPv6 SEND messages.
