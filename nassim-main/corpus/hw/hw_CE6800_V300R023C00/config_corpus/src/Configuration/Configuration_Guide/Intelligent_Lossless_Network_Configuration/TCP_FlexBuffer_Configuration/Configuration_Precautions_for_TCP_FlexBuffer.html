WRED profile resources occupied by TCP FlexBuffer

For the CE8850-HAM/CE8850-SAN/CE8851-32CQ8DQ-K/CE8851-32CQ8DQ-P:

TCP FlexBuffer occupies four WRED profiles. After TCP FlexBuffer is enabled on a device, the number of WRED profiles that can be configured on the device decreases by four.

For the CE6820H-48S6CQ/CE6820S-48S6CQ/CE6820H-48S6CQ-K/CE6863H-48S6CQ/CE6863H-48S6CQ-K/CE6881H-48S6CQ/CE6881H-48S6CQ-K/CE6881H-48T6CQ/CE6881H-48T6CQ-K:

TCP FlexBuffer occupies nine WRED profiles. After TCP FlexBuffer is enabled on a device, the number of WRED profiles that can be configured on the device decreases by nine.

For the CE6860-HAM/CE6860-SAN/CE6866-48S8CQ-K/CE6866-48S8CQ-P:

TCP FlexBuffer occupies four WRED profiles. After TCP FlexBuffer is enabled on a device, the number of WRED profiles that can be configured on the device decreases by four.
