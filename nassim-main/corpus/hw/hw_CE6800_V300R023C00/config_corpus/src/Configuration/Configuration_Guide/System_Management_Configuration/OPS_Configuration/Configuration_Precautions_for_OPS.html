Feature Requirements

Series

Models

1. Maintenance assistants are classified as command assistants or Python script assistants. A maximum of 100 maintenance assistants can be configured.

2. A single script assistant can be configured with one to eight conditions. To configure multiple conditions, use the AND, OR, and NOT logical operators between the conditions. A single command assistant can be configured with only one condition.

3. If the number of conditions subscribed by all maintenance assistants (including preset assistants) exceeds 200, no maintenance assistant can be created.

4. A maximum of 10 subscribed events in a Python script can be matched at the same time.

5. A maximum of 100 user-defined environment variables and 100 script variables are allowed on the device.

6. Due to the startup timing of the distributed system, an OPS maintenance assistant may cause events that occur during the startup phase. If high reliability is required for the events during the startup phase, you are advised to use a Python script assistant to check the device status in the subscription phase for smooth restoration, or reconfigure a command assistant in 23.0 and later versions.
