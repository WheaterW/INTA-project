Verifying the Configuration

Procedure

  • Run the display isis ipv6 bfd [ process-id | vpn-instance vpn-instance-name ] session { all | peer peer-addr | interface { interface-name | interface-type interface-number } } command to check information about BFD for IPv6 IS-IS sessions.
  • Run the display isis ipv6 bfd [ process-id | vpn-instance vpn-instance-name ] interface command to check information about the interfaces enabled with BFD for IPv6 IS-IS.
