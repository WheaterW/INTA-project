Verifying the ERPSv1 Configuration

Procedure

  1. Run the display erps [ ring ring-id ] [ verbose ] command to check device ports added to an ERPS ring and ERPS ring configurations.
  2. Run the display erps interface interface-type interface-number [ ring ring-id ] command to check the physical status of ports added to an ERPS ring.
