Verifying the Configuration

Prerequisites

PM has been configured.

Procedure

  • Run the display pm brief command to check brief PM information.
  • Run the display pm statistics-task [ task-name ] command to check information about a performance statistics task.
  • Run the display pm measure-info [ instance-type instance-type ] command to check statistics about an instance.
  • Run the display pm statistics task-name data-index index [ instance-type instance-type-name [ measure measure-name | instance { vpn-instance-name } &<1-8> ] * ] command to view PM statistics.
  • Run the display pm statistics-file [ task-name ] command to check the list of statistics files.
