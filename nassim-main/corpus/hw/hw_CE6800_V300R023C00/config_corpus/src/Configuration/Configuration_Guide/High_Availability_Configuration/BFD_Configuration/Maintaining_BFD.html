Clearing BFD Statistics

In BFD maintenance, you can clear unneeded BFD statistics, monitor the BFD operating status, and debug BFD if a fault occurs.

To delete unneeded BFD statistics or improve new statistics lookup efficiency, run the reset bfd statistics command to clear existing statistics.

Deleted BFD statistics cannot be restored. Exercise caution when you run the reset bfd statistics command.

Table 2 Clearing BFD Statistics

Operation

Command

Clear BFD statistics.

reset bfd statistics { all | discriminator discr-value }
