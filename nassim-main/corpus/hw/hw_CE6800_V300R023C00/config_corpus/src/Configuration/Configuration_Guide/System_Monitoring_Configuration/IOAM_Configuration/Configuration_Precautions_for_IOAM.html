Because content needs to be added to the packet header, ensure path MTU compatibility.

1. In hop-by-hop direct-export mode, only a 20-byte IOAM header is added to a packet.

2. If data is reported by a termination node:

(1) In trace mode, the packet length is increased by (8 + 12) + 32 x N bytes (N indicates the number of hops).

(2) In edge-to-edge mode, a transit node does not add its metadata to a packet, and only the ingress node adds 32 bytes (8 + 8 + 16) to the packet.
