Verifying the Configuration

Procedure

  • Run the display info-center [ statistics ] command to check IM statistics.
  • Run the display info-center channel [ channel-number | channel-name ] command to check the information channel.
  • Run the display debugging packet all command to check the running status of all protocol packets in displayed debugging information.
  • Run the display debugging timeout command to check the timeout period of the system debugging function.
