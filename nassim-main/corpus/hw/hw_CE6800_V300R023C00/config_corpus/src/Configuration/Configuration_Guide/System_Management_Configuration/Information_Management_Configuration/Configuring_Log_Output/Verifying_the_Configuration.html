Verifying the Configuration

Procedure

  • Run the display info-center [ statistics ] command to check IM statistics.
  • Run the display info-center channel [ channel-number | channel-name ] command to check the information channel.
  • Run the display logbuffer command to check information in the log buffer.
  • Run the display logfile path [ offset ] command to check information recorded in the log file.
  • Run the display logfile [ log | diagnose ] list starttime starttime-value endtime endtime-value command to check the list of log files generated in a specified period.
