Configuring IPv6 IS-IS SRLG FRR

Prerequisites

Before configuring IPv6 IS-IS SRLG FRR, you have completed the following task:

  • Configure basic IPv6 IS-IS functions.

Procedure

  1. Enter the system view.

    system-view

  2. Enter the interface view.

    interface interface-type interface-number

  3. Switch the interface working mode to Layer 3.

    undo portswitch

    Determine whether to perform this step based on the current interface working mode.

  4. Add the interface to an SRLG.

    isis srlg srlg-value

    By default, an interface is not added to any SRLG.

  5. (Optional) Set the solution of selecting a backup path to SRLG-disjoint path first.

    quit
isis process-id
ipv6 frr
loop-free-alternate [ level-1 | level-2 | level-1-2 ]
tiebreaker srlg-disjoint preference preference [ level-1 | level-2 | level-1-2 ]

  6. Commit the configuration.

    commit

Verifying the Configuration

  • Run the display isis [ process-id ] srlg { srlgGroupId | all } command to check information about SRLGs.
