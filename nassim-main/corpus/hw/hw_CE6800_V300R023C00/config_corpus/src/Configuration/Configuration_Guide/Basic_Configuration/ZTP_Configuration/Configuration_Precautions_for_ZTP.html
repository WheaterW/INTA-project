ZTP can be deployed through the management interface or service interface. However, deploying through the management interface is recommended. If the service interface is used for deployment, the device may be inaccessible due to triple-plane isolation (isolation between the service plane, management plane, and control plane). Therefore, you are advised to deploy ZTP in a secure networking environment.
