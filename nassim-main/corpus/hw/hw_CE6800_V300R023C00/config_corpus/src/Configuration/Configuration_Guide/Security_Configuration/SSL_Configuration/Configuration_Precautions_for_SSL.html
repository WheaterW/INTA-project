Create an SSL policy. The default DH modulus is 3072, and the value can be 2048, 3072, or 4096. By default, ed25519, ed448, rsa-pss-pss-sha256, rsa-pss-pss-sha384, rsa-pss-pss-sha512, rsa-pss-rsae-sha256, rsa-pss-rsae-sha384 and rsa-pss-rsae-sha512 are enabled for signature algorithms, which can be configured separately to enhance security.

After an SSL policy is created, if the SSL handshake fails because the signature algorithm does not match or the DH modulus length is too long, run the diffie-hellman modulus command to adjust the DH modulus length or run the signature algorithm-list command to adjust the signature algorithm.
