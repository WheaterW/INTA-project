Plug-and-Play Storage Service

If two iNOF reflectors exist in an iNOF system, they must be specified as each other's client. In addition, other clients configured using the peer reflect-client command on the two reflectors must be the same. In this way, the two reflectors can back up each other to enhance the reliability of the iNOF system.
