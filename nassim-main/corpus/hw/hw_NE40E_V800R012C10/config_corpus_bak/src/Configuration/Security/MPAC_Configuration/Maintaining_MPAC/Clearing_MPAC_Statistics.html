Clearing MPAC Statistics

This section describes how to clear MPAC statistics.

Context

MPAC statistics cannot be restored after being cleared. Exercise caution when running the reset service-security counters { ipv4 | ipv6 } [ security-policy-name ] command.

Procedure

  • Run the reset service-security counters { ipv4 | ipv6 } [ security-policy-name ] command to clear MPAC statistics.
