Verifying the DAA Configuration

After configuring DAA, verify the DAA configuration.

Procedure

  • Run the display value-added-service policy command to check information about value-added service policies.
  • Run the display value-added-service user command to check information about all users using value-added services.
  • Run the display value-added-service user user-id command to check information about a specified user using value-added services.
