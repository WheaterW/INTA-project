Verifying the Configuration of the Console User Interface

After configuring the console user interface, verify the configuration.

Prerequisites

The console user interface has been configured.

Procedure

  • Run the display users [ all ] command to check user login information about user interfaces.
  • Run the display user-interface console 0 command to check physical attributes and configurations of the user interface.

    In VS mode, this command is supported only by the admin VS.

  • Run the display local-user command to check the local user list.
  • Run the display access-user command to check information about users that pass AAA authentication.
