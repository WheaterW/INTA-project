Verifying the Configuration of a DHCP Server

After configuring the DHCP server, verify the configuration.

Prerequisites

The DHCP server has been configured.

Procedure

  • Run the display ip pool command to check the address pool configurations and address statistics.
  • Run the display dhcp server database command to check the saved DHCP server data.
