Configuring the SyncE Function

This section describes how to configure the SyncE function.

Usage Scenario

On the IP RAN shown in Figure 1, clock synchronization needs to be achieved between gNodeBs. The Atom GNSS timing solution can be deployed as follows to allow clock synchronization: Insert an Atom GNSS module into an ASG so that the Atom GNSS module can synchronize clock signals with the GNSS and the ASG can synchronize clock signals with the Atom GNSS module. Then, configure the SyncE function to allow transmission of clock signals to downstream devices and then to NodeBs. In this manner, network-wide clock synchronization is achieved.
