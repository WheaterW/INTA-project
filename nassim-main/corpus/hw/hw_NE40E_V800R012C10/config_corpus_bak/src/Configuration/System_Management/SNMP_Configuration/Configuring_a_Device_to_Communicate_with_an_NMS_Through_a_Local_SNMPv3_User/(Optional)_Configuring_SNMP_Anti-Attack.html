(Optional) Configuring SNMP Anti-Attack

To defense against a user's attack on other users' passwords, configuring the SNMPv3 blacklist function to improve security.

Procedure

  1. Run the system-view command to enter the system view.
  2. Run the undo snmp-agent blacklist ip-block disable command to enable the IP address blacklist function.
  3. Run the undo snmp-agent blacklist user-block disable command to enable the user blacklist function.
  4. Run the commit command to commit the configuration.
