Activating Distributed Patch File

This section describes how to activate distributed patch files.

Applicable Environment

After a board is installed, activate the distributed patch file so that you can perform further configurations on the board.

In VS mode, this command is supported only by the admin VS.

Procedure

  1. Run system-view

    The system view is displayed.

  2. Run pnp-package enable

    The distributed patch file is activated.
