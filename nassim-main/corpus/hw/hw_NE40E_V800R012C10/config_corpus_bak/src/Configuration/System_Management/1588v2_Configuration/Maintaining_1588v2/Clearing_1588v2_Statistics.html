Clearing 1588v2 Statistics

This section describes how to clear 1588v2 statistics.

Context

ACL statistics cannot be restored after being cleared. Exercise caution when running the reset command.

After you confirm to clear 1588v2 statistics, run the following command.

Procedure

  1. Run the reset ptp statistics { all | interface interface-type interface-number } command in the user view to clear statistics about 1588v2 packets.
