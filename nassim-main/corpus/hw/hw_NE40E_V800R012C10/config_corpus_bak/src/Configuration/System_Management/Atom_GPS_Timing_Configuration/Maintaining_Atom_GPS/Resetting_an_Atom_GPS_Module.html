Resetting an Atom GPS Module

This section describes how to reset an Atom GPS module.

Context

After the Atom GPS module is reset, the clock source is switched or the clock enters the holdover state. After the module is locked again, the device can trace the module again.

Perform the following operations on the Router where the Atom GPS module resides:

Procedure

  1. Run the system-view command to enter the system view.
  2. Run the interface interface-type interface-number command to enter the interface view.
  3. Run the reset smart-clock command to reset the Atom GPS module on the interface.
