Configuring VPLS Between SPEs

The VPLS configuration between SPEs is similar to the configuration of common VPLS.

Context

The VPLS configuration between SPEs is similar to the configuration of common full-mesh VPLS. The configuration on the SPE is similar to the common VPLS configuration. For details, see Configuring LDP VPLS.
