Verifying the Configuration

After configuring BGP4+ GR helper, check the BGP4+ GR status.

Prerequisites

The BGP4+ GR helper has been configured.

Procedure

  • Run the display bgp ipv6 peer verbose command to check the BGP4+ GR status.
