Verifying the Route-Policy Configuration

After configuring a route-policy, verify information about the route-policy.

Prerequisites

The route-policy has been configured.

Procedure

  1. Run the display route-policy [ route-policy-name ] command to check information about the route-policy.
  2. (Optional) Run the reset route-policy route-policy-name counters command to reset route-policy counters.
