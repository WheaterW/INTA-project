Checking the Configuration

After configuring MC-LMSP, you can check the LMSP group's working mode, working interface, switchback time, and interface status.

Procedure

  1. Run the display aps group aps-group-id command to check configurations of an LMSP group.
  2. Run the display cpos-trunk trunk-id command to check configurations of the CPOS-Trunk interfaces.
