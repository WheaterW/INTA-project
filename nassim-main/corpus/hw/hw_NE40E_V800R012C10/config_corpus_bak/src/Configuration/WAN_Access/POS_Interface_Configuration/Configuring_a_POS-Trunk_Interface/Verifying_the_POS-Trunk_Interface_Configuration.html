Verifying the POS-Trunk Interface Configuration

After completing the basic configurations of a POS-Trunk interface, check the configurations.

Prerequisites

Basic configurations have been completed on a POS-Trunk interface.

Procedure

  • Run the display pos-trunk trunk-id command to check the configurations of a POS-Trunk interface.
