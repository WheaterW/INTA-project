Verifying the Configuration

After configuring a single-chassis LMSP group, the LMSP group's working mode, working interface, WTR time, and interface status can be viewed.

Procedure

  1. Run the display aps group aps-group-id command to check configurations of an LMSP group.
  2. Run the commands below to check configurations of trunk interfaces.

    • Run the display pos-trunk trunk-id command to check configurations of the POS-Trunk interfaces.
    • Run the display cpos-trunk trunk-id command to check configurations of the CPOS-Trunk interfaces.
