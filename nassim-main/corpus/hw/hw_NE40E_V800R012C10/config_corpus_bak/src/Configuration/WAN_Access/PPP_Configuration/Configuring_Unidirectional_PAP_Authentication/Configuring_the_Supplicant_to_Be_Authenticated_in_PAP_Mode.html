Configuring the Supplicant to Be Authenticated in PAP Mode

This section describes how to configure the supplicant to be authenticated in PAP mode.

Procedure

  1. Run system-view

    The system view is displayed.

  2. Run interface interface-type interface-number

    The interface view of the supplicant is displayed.

  3. Run ppp pap local-user user-name password { simple password2 | cipher { password1 | password2 } }

    The interface is configured to send the local PAP username and password to the authenticator when the local end is authenticated by the authenticator in PAP mode.

  4. Perform the following steps to restart the interface:
    1. Run the shutdown command to shut down the interface.
    2. Run the commit command to commit the configuration.
    3. Run the undo shutdown command to restart the interface.

    After completing the configuration, run the shutdown and undo shutdown commands to restart the interface for the configuration to take effect.

    After running the shutdown command during an interface restart, run the commit command to commit the configuration. Otherwise, the configuration does not take effect.

  5. Run commit

    The configuration is committed.
