Clearing ATM Interface Statistics

You can run the reset commands to clear interface statistics before recollecting traffic statistics on the interface.

Context

Interface statistics cannot be restored after being cleared. Exercise caution when running reset commands.

To clear interface statistics displayed on the NMS or in the display interface command output, run the following commands in the user view. After the statistics are cleared, the interface starts to collect statistics again.

See the relevant NMS manual to learn about how to view the interface traffic statistics on the NMS.

Procedure

  • Run the reset counters interface [ atm [ interface-number ] ] command to clear statistics displayed in the display interface command output.
  • Run the reset counters if-mib interface [ atm [ interface-number ] ] command to clear interface statistics displayed on the NMS.
  • Run the reset atm interface [ atm interface-number ] command to clear PVC statistics of an ATM interface.
