Clearing NAT Session Entries

This section describes how to use the reset command to clear NAT session entries.

Context

To delete unnecessary NAT session entries, run the reset nat session table command. This allows new NAT session entries to be created for fault location and rectification.

Services will be interrupted after NAT session entries are cleared. Exercise caution when performing this operation.

In VS mode, this configuration process is supported only by the admin VS.

Procedure

  • After determining the NAT session entries to be cleared, run reset nat session table
