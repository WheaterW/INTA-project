Checking the Configurations

After configuring basic MAP-E functions, verify the configurations.

Prerequisites

The configurations of basic MAP-E functions are complete.

Procedure

  • Run the display map instance [ instance-name ] [ verbose ] command to check the configurations of a MAP-E instance.
  • Run the display map rule [ rule-name ] command to check the basic mapping rule (BMR) configurations of a MAP-E instance.
