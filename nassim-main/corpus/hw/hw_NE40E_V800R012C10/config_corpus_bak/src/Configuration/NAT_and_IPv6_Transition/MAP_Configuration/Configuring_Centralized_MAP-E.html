Configuring Centralized MAP-E

Centralized Scenario

In a centralized scenario, the MAP-BR and BRAS reside on different devices. The BRAS functions deliver MAP addresses and mapping rules to MAP-CEs in DHCPv6 IA_PD mode. Router A functioning as the MAP-BR resides on the edge of a MAP domain and allows MAP-CEs to access the public IPv4 network through the IPv6 network that is within the MAP domain. The MAP-CEs use each other's public IPv4 address to communicate through the MAP-BR.
