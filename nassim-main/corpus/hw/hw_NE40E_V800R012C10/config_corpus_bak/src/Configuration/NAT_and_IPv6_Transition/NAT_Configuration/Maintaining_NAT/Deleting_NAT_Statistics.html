Deleting NAT Statistics

Run the following reset command in the user view to delete NAT statistics.

Context

Before you collect NAT statistics within a specified period of time, delete existing NAT statistics using the reset nat statistics command.

Once the statistics are deleted, they cannot be restored. Exercise caution when using the command.

Procedure

  • After confirming to delete NAT service statistics, run the reset nat statistics command.
