Verifying the L2TP HQoS Configuration

Through L2TP group configurations, you can verify the configured QoS profile and scheduling mode of L2TP.

Prerequisites

All L2TP HQoS configurations are complete.

Procedure

  • Run the display l2tp-group group-name command to check the configurations of all L2TP groups or a specified one.
