Verifying the DHCPv4 Server Group Configuration

After configuring DHCPv4 server groups, verify the configurations of these DHCPv4 server groups.

Prerequisites

DHCPv4 server groups have been configured.

Procedure

  • Run the display dhcp-server group [ group-name ] command to check the configurations of all DHCPv4 server groups or a specified one.
  • Run the display cpu-defend whitelist session-car dhcp-server statistics slot slot-id command to check DHCP protocol whitelist session-CAR statistics on a specified interface board.
