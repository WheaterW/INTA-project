Verifying the Configuration

After configuring dual-device IGMP snooping hot backup, verify the configurations.

Prerequisites

Dual-device IGMP snooping hot backup has been configured.

Procedure

  • Run the display remote-backup-profile [ profile-name ] command to check information about a specified RBP.
  • Run the display remote-backup-service [ service-name [ verbose ] ] command to check information about a specified remote backup service (RBS).
