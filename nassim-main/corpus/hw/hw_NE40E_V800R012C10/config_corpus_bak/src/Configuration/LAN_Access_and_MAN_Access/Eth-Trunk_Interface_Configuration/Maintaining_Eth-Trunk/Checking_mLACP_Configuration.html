Checking mLACP Configuration

Context

You can run the following commands in any view to check the mLACP configuration during routine maintenance.

Procedure

  1. Run the display iccp group group-id command to check information about an ICCP redundancy group.
  2. Run the display mlacp iccp-status group group-id command to check mLACP information in an ICCP redundancy group.
  3. Run the display mlacp eth-trunk eth-trunk-id command to check detailed information about Eth-Trunk interfaces added to mLACP.
