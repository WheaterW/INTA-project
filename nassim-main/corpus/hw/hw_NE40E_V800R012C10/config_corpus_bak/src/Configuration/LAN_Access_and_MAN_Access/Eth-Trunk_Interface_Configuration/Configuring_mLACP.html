Configuring mLACP

Usage Scenario

mLACP provides similar functions to E-Trunk, as shown in Figure 1. In multi-chassis scenarios, the local device cannot obtain the configuration and negotiation parameters of a peer device that has Eth-Trunk in LACP mode configured, so master/backup protection cannot be implemented. mLACP can be used to resolve this issue. It synchronizes LACP configuration and status information between dual-homed devices through a reliable ICCP channel, making master/backup protection possible.
