Enabling PIM-SM

To enable a Router interface to establish PIM neighbor relationships with other Routers, enable PIM-SM.

Procedure

  1. Run system-view

    The system view is displayed.

  2. Run interface interface-type interface-number

    The interface view is displayed.

  3. Run pim sm

    PIM-SM is enabled.

    Running the undo pim sm command will delete PIM neighbor relationships on the interface and interrupt the multicast service running on the interface.

  4. Run commit

    The configuration is committed.
