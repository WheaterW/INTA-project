Verifying the P2MP TE FRR Configuration

After configuring P2MP TE FRR, verify P2MP TE FRR information.

Prerequisites

P2MP TE FRR has been configured.

Procedure

  • Run the display mpls te p2mp tunnel frr [ tunnel-name ][ lsp-id ingress-lsr-id session-id local-lsp-id [ s2l-destination leaf-address ] ] command to check bypass tunnel attributes.
