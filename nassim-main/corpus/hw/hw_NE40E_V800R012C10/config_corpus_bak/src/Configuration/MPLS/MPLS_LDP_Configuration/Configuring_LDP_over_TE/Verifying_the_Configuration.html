Verifying the Configuration

After configuring LDP over TE, you can view information about an LDP LSP on the ingress.

Prerequisites

LDP over TE has been configured.

Procedure

  • Run the display mpls ldp lsp [ destination-address mask-length ] command to check information about the LDP LSP on the ingress.
