Configuring SRv6 BE Services

After SRv6 BE services are configured, they can be forwarded based on network slices.

Context

Configuring SRv6 BE services includes configuring BGP L3VPN, EVPN L2VPN, EVPN L3VPN, and public IP addresses. For details, see Configuring SRv6 BE (IS-IS).
