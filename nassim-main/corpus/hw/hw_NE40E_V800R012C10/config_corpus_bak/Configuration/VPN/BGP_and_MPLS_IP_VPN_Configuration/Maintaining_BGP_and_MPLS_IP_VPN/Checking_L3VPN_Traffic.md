Checking L3VPN Traffic
======================

This section describes how to collect traffic statistics
on the interface at the user side of the VPN. Note that traffic statistics
are collected only after the L3VPN traffic statistics function is
enabled.

#### Procedure

* Run the [**display
  traffic-statistics vpn-instance**](cmdqueryname=display+traffic-statistics+vpn-instance) *vpn-instance-name* command to check the statistics about the specified L3VPN traffic.
