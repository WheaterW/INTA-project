Verifying the Configuration
===========================

After configuring an MPLS-based ACL, verify the configuration.

#### Prerequisites

An MPLS-based ACL has been configured.


#### Procedure

* Run the [**display acl**](cmdqueryname=display+acl+name+all) { *acl-number* | **name** *acl-name* | **all** } command to check MPLS-based ACL configurations.
* Run the [**display time-range**](cmdqueryname=display+time-range+all) { *time-name* | **all** } command to check the configuration and status within a specified time range or all time ranges.
