Enabling IPv6
=============

You can perform IPv6-related configurations on an interface only when IPv6 is enabled in the interface view.

#### Procedure

1. Run [**system-view**](cmdqueryname=system-view)
   
   
   
   The system view is displayed.
2. Run [**interface**](cmdqueryname=interface) *interface-type* *interface-number*
   
   
   
   The view of an interface on which IPv6 is to be enabled is displayed.
3. Run [**ipv6 enable**](cmdqueryname=ipv6+enable)
   
   
   
   IPv6 is enabled on the interface.
4. Run [**commit**](cmdqueryname=commit)
   
   
   
   The configuration is committed.
