Verifying the Configuration of Multicast over P2MP TE Tunnels
=============================================================

After configuring multicast over point-to-multipoint (P2MP) traffic engineering (TE) tunnels, verify the configuration.

#### Prerequisites

The multicast over a P2MP TE tunnels configuration is complete.
#### Procedure

* Run the [**display pim**](cmdqueryname=display+pim) [ **vpn-instance** *vpn-instance-name* | **all-instance** ]  **rp-info** command to check information about the PIM routing table.
