Verifying the Configuration of BFD for IPv6 PIM

After configuring BFD for IPv6 PIM, verify statistics about and configurations of BFD for IPv6 PIM sessions.

Prerequisites

All the configurations of BFD for IPv6 PIM are complete.

Procedure

  • Run the display pim ipv6 bfd session statistics command to check information of BFD sessions.
  • Run the display pim ipv6 bfd session [ interface interface-type interface-number | neighbor ipv6-link-local-address ] * command to check configurations of BFD sessions.
