Deleting Statistics About IGMP Messages

If you need to re-collect the statistics about IGMP messages, delete the existing statistics.

Context

Deleted statistics about IGMP messages cannot be restored. Therefore, exercise caution when running the reset igmp command.

Procedure

  1. Run the reset igmp [ vpn-instance vpn-instance-name | all-instance ] control-message counters [ interface interface-type interface-number ] [ message-type { query | report } ] command in the user view to delete statistics about IGMP messages.
