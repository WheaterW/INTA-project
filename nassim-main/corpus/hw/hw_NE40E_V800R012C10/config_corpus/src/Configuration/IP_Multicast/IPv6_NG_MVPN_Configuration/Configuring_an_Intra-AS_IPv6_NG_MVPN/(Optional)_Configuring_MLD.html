(Optional) Configuring MLD

Configuring MLD on the interfaces connecting a multicast device to a user network segment allows the device to manage multicast group members on the network segment.

Context

In the TCP/IP protocol suite, Multicast Listener Discovery (MLD) manages IPv6 multicast group members. It sets up and maintains multicast group member relationships between IP hosts and neighboring multicast routers.

Procedure

  1. Run system-view

    The system view is displayed.

  2. Run interface interface-type interface-number

    The interface view is displayed.

    interface-type interface-number specifies the interface connecting to a user host.

  3. Run pim ipv6 sm

    IPv6 PIM-SM is enabled.

  4. Run mld enable

    MLD is enabled.

  5. Run commit

    The configuration is committed.

Follow-up Procedure

Configure an MLD version, add interfaces to a static multicast group, and specify the multicast groups to which interfaces are allowed to be added. For details, see Configuring Basic MLD Functions.
