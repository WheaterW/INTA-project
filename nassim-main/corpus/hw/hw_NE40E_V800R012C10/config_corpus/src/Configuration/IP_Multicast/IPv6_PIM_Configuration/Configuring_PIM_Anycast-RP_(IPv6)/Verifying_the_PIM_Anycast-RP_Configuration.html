Verifying the PIM Anycast-RP Configuration

After configuring PIM Anycast-Rendezvous Point (RP), verify the Anycast-RP configuration.

Prerequisites

PIM Anycast-RP has been configured.

Procedure

  • Run the display pim ipv6 rp-info command to check RP information.
  • Run the display pim ipv6 routing-table command to check the IPv6 PIM routing table.
