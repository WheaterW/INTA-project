Verifying the Configuration

After associating a VRRP6 group with a VRRP6-disabled interface or a BFD session, verify the configurations.

Prerequisites

VRRP6 association has been configured.

Procedure

  • Run the display vrrp6 interface interface-type interface-number [ vrid virtual-id ] verbose command to check the detailed information about the VRRP6 group.
