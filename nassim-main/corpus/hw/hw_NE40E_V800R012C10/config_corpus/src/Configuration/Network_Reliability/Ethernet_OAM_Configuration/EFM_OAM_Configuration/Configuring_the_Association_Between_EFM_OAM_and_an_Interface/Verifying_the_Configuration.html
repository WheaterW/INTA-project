Verifying the Configuration

After associating EFM OAM with an interface, verify the configurations.

Prerequisites

EFM OAM has been associated with an interface.

Procedure

  • Run the display efm { all | interface interface-type interface-number } command to check the EFM OAM status of interfaces.
