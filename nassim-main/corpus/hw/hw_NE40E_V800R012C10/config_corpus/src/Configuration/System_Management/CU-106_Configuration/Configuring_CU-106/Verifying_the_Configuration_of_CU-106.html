Verifying the Configuration of CU-106

After configuring the CU-106 function, verify the configuration.

Procedure

  • Run the display ptp all [ state | config ] command to view the time synchronization status and configurations of device operation.
  • Run the display ptp interface interface-type interface-number command to view the time synchronization information on a device interface.
