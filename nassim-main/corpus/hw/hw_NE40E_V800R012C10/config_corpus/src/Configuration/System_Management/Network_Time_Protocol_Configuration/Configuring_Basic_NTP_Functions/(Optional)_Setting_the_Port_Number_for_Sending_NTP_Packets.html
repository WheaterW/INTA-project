(Optional) Setting the Port Number for Sending NTP Packets

Procedure

  1. Run the system-view command to enter the system view.
  2. Run the ntp-service source-port portNum command to configure the port number for sending NTP packets. If source-port is not configured, the port number is randomly allocated.
  3. Run the commit command to commit the configuration.
