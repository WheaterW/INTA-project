NE40E-M2E/NE40E-M2F/NE40E-M2H/NE40E-M2K/NE40E-M2K-B

NETCONF has a limitation on the maximum number of imported packets. The maximum size of the configuration data imported through the URL cannot exceed 30 MB.

The following operations are involved:

1. In the edit-config operation, the value of source is url.

2. In the copy-config operation, the value of source is url.

The size of target data in the URL cannot exceed 30 MB.

1. If the capability set carried in the Hello packet sent by the peer contains extended capabilities, the device replies with a schema packet.

2. The exchange capability set is advertised to identify the mode of processing high-specification data.

3. The sync capability set is advertised to determine whether the packet returned by edit-config carries a flow ID.
