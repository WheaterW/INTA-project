Monitoring LLDP Status

This section describes how to use display commands to monitor LLDP status.

Context

In routine maintenance, you can run the following commands in any view to check the LLDP status.

Procedure

  • Run the display lldp statistics [ interface interface-type interface-number ] command to check statistics about LLDP packets transmitted by all interfaces or a specified interface.
