Verifying the IPoE Access Configuration

After configuring IPoE access, you can view information about IPoE access.

Procedure

  • Run the display access-user command to check information about online users. To view information about one or more specified users, configure parameters as required.
  • Run the display web-auth-server configuration command to check the web authentication server configuration.
  • Run the display domain command to check the configurations of all domains or a specific one.
  • Run the display acl command to check the configurations of all ACLs or a specific one.
  • Run the display interface command to check the VE interface status.
