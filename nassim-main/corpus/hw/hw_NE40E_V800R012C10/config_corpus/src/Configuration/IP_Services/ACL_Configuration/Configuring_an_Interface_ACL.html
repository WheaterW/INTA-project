Configuring an Interface ACL

An interface ACL defines rules to filter packets.

Usage Scenario

Figure 1 Configuring an interface ACL

Interfaces 1 and 2 in this example represent GE 0/1/0 and GE 0/2/0, respectively.


As shown in Figure 1, an ACL based on GE 0/1/0 is created on DeviceA to allow DeviceA to permit all packets sent from Network A to the Internet and deny all packets sent from Network B to the Internet.
