Monitoring the DHCP Relay Operating Status

This section describes how to monitor the DHCP relay operating status.

Context

You can run the following commands in any view to check the DHCP relay operating status in routine maintenance.

Procedure

  • Run the display dhcp relay statistics [ interface interface-type interface-number ] command to check DHCP relay statistics.
  • Run the display dhcp relay address [ interface interface-type interface-number ] command to check the address configuration of the DHCP relay agent.
