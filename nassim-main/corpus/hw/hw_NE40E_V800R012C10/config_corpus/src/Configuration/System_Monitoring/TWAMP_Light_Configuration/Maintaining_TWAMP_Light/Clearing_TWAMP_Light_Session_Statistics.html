Clearing TWAMP Light Session Statistics

This section describes how to clear TWAMP Light session statistics.

Context

TWAMP Light session statistics cannot be restored after being cleared. Exercise caution when clearing the statistics.

Procedure

  • Run the reset twamp-light statistics { all | test-session session-id } command to clear TWAMP Light session statistics.
