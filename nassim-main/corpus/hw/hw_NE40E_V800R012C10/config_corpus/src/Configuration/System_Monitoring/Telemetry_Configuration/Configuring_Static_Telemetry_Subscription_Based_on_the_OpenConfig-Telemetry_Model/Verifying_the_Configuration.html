Verifying the Configuration

After configuring static telemetry subscription to the sampled data or a customized event, check information about the sampling sensor group, destination group, and subscription.

Prerequisites

All configurations of static telemetry subscription are complete.

Procedure

  • Run the display telemetry sensor [ sensor-name ] command to check the sampling sensor information.
  • Run the display telemetry destination [ dest-name ] command to check information about the destination group.
  • Run the display telemetry subscription [ subscription-name ] command to check the subscription information.
  • Run the display telemetry sensor-path command to check the sampling path of a telemetry sensor.
