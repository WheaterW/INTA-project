Verifying the Configurations

After configuring mLDP P2MP FRR, check P2MP multicast LSP information. The command output shows FRR LSP information.

Prerequisites

mLDP P2MP FRR has been configured.

Procedure

  • Run the display mpls multicast-lsp protocol mldp p2mp command to check P2MP multicast LSP information, including FRR LSP information.
