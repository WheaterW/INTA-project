Configuring Route Exchange Between PEs and Access-Side Devices

PEs and access-side devices can communicate through BGP, static routes (including default routes), or IGP. Configure route exchange between PEs and access-side devices according to the network plan.

Procedure

  1. Deploy BGP, IGP, or static routes (including default routes) between PEs and access-side devices based on the network plan. For configuration details, see Configuring Route Exchange Between PEs and CEs or Configuring Route Exchange Between PEs and CEs.
