Verifying the Configuration

Prerequisites

Basic APN6 functions have been configured.

Procedure

  1. Run the display this command in the APN view to check the APN6 configuration.
  2. Run the display apn-id-ipv6 instance [ name instance-name ] command to check detailed information about APN ID instances.
