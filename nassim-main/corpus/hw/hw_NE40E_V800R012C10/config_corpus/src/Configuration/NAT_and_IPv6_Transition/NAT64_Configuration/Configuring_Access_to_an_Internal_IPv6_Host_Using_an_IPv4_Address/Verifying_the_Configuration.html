Verifying the Configuration

You can verify the configuration of the internal server function in a NAT64 instance.

Procedure

  • Run the display nat64 instance [ instance-name ] command to check the configuration of a NAT64 instance.
  • Run the display nat server-map [ dynamic | static ] [ ip ip-address | port port-number | vpn-instance vpn-instance-name | slot slot-id ] * command to check server-map entry information of an internal server.
