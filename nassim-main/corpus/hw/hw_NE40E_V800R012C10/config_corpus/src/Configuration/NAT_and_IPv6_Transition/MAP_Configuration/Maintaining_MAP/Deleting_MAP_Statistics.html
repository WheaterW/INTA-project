Deleting MAP Statistics

This section describes how to delete statistics about forwarded and dropped MAP packets.

Context

MAP statistics cannot be restored after being deleted. Exercise caution when running the following command.

Procedure

  • After you confirm to delete MAP statistics, run the reset map statistics [ rule rule-name ] [ slot slot-id ] command in the user view.
