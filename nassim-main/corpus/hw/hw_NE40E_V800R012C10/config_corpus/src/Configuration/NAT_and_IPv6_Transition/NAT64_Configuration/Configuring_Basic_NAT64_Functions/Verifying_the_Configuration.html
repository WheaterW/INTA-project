Verifying the Configuration

After basic NAT64 functions are configured, you can verify the basic NAT64 configuration.

Procedure

  • Run the display service-location service-location-id command to check the service-location configuration.
  • Run the display service-instance-group service-instance-group-name command to check configuration information of a service instance group.
  • Run the display nat64 instance [ instance-name ] command to check the configuration of a NAT64 instance.
