Verifying the Configuration

After configuring MACsec, you can run display commands to check MACsec configurations.

Procedure

  1. Run the display macsec statistics interface { interface-name | interface-type interface-number } command to check statistics about data packets protected by MACsec.
  2. Run the display mka interface { interface-name | interface-type interface-number } command to check MKA session information.
