Configuration Precautions for Weak Password Dictionary

#### Feature Requirements

**Table 1** Feature requirements
| Feature Requirements | Series | Models |
| --- | --- | --- |
| - The maximum size of a weak password file is 1 MB.  - A weak password can contain a maximum of 128 characters.  - Weak passwords are stored in the database in plaintext and can be queried.  - A maximum of 1000 weak passwords can be loaded on the device. | NE40E-M2 | NE40E-M2E/NE40E-M2F/NE40E-M2H/NE40E-M2K/NE40E-M2K-B |
