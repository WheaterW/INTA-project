Verifying the Configuration
===========================

After multicast shaping is configured, you can view information about the online users with specified IDs and the bandwidth consumed by them and applications of the QoS profile.

#### Prerequisites

Multicast shaping has been configured.


#### Procedure

1. Run the [**display multicast-list**](cmdqueryname=display+multicast-list) [ *group-list-name* ] command to check the configuration of the multicast program list.
