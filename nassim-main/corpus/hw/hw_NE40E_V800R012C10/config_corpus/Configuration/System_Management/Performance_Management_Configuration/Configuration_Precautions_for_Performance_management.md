Configuration Precautions for Performance management
====================================================

Configuration Precautions for Performance management

#### Feature Requirements

**Table 1** Feature requirements
| Feature Requirements | Series | Models |
| --- | --- | --- |
| Patch operations cannot be performed if the system CPU usage exceeds 70%. Otherwise, the patch fails to be loaded.  Install the patch after the CPU usage decreases to less than 70%. | NE40E-M2 | NE40E-M2E/NE40E-M2F/NE40E-M2H/NE40E-M2K/NE40E-M2K-B |
| Do not install the patch if the total memory usage of the system and the estimated memory usage for patch installation exceeds 90%. | NE40E-M2 | NE40E-M2E/NE40E-M2F/NE40E-M2H/NE40E-M2K/NE40E-M2K-B |
| KPI files generated by the KPI diagnosis function are not independently saved by VS. | NE40E-M2 | NE40E-M2E/NE40E-M2F/NE40E-M2H/NE40E-M2K/NE40E-M2K-B |
| Configurations cannot be concurrently committed for Admin-VS and non-Admin-VS VSs.  Configurations can be concurrently committed for multiple non-Admin-VS VSs. However, if configurations that affect Admin-VS are modified for a non-Admin-VS VS, configurations cannot be committed for other non-Admin-VS VSs during configuration commission for this non-Admin-VS VS. | NE40E-M2 | NE40E-M2E/NE40E-M2F/NE40E-M2H/NE40E-M2K/NE40E-M2K-B |
| A non-admin VS does not support fast startup in DAT mode. | NE40E-M2 | NE40E-M2E/NE40E-M2F/NE40E-M2H/NE40E-M2K/NE40E-M2K-B |
| A maximum of 21 users are allowed to deliver configurations on a device. Other users can only query configurations. The users in non-admin VSs are in preemption mode. | NE40E-M2 | NE40E-M2E/NE40E-M2F/NE40E-M2H/NE40E-M2K/NE40E-M2K-B |
| IP FPM measurement flags cannot be configured for VSs except VS0. | NE40E-M2 | NE40E-M2E/NE40E-M2F/NE40E-M2H/NE40E-M2K/NE40E-M2K-B |
| DCN can be enabled only in the Admin-VS. | NE40E-M2 | NE40E-M2E/NE40E-M2F/NE40E-M2H/NE40E-M2K/NE40E-M2K-B |
| GRESM labels in the system are unique and cannot conflict with each other even in different VSs. Do not configure the same static GRESM label for different VSs. | NE40E-M2 | NE40E-M2E/NE40E-M2F/NE40E-M2H/NE40E-M2K/NE40E-M2K-B |
| The group-mac of L2PT and BPDU tunnels can be configured only in the admin VS. | NE40E-M2 | NE40E-M2E/NE40E-M2F/NE40E-M2H/NE40E-M2K/NE40E-M2K-B |
| The BFD local discriminator (MD) is unique on a device. Different VSs cannot be configured with the same local discriminator. | NE40E-M2 | NE40E-M2E/NE40E-M2F/NE40E-M2H/NE40E-M2K/NE40E-M2K-B |
| EVPN supports multiple VSs, but the ESI-ID must be unique on a device. Do not configure the same ESI-ID for different non-VS0 VSs. | NE40E-M2 | NE40E-M2E/NE40E-M2F/NE40E-M2H/NE40E-M2K/NE40E-M2K-B |
