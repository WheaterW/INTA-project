Verifying the Configuration
===========================

After configuring a user ACL, verify the configuration.

#### Prerequisites

All configurations of the user ACL are configured.


#### Procedure

* Run the [**display acl**](cmdqueryname=display+acl+name+all) { *acl-number* | **name** *acl-name* | **all** } command to check user ACL configurations.
* Run the [**display time-range**](cmdqueryname=display+time-range+all) { *time-name* | **all** } command to check the configuration and status within a specified time range or all time ranges.
* Run the [**display acl url-address**](cmdqueryname=display+acl+url-address) [ *url-address* ] command to check information about all the IP addresses that are associated with domain names and meet the conditions.
