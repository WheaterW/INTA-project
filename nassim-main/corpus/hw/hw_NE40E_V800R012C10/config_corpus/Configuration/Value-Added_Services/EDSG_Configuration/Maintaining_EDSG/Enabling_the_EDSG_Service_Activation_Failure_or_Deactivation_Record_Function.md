Enabling the EDSG Service Activation Failure or Deactivation Record Function
============================================================================

For fault diagnosis and locating, you can enable the EDSG
service activation failure or deactivation record function.

#### Procedure

* In the system view, run [**service activate-fail-record**](cmdqueryname=service+activate-fail-record)
  
  
  
  The EDSG service activation failure record function is enabled.
* In the system view, run [**service deactivate-record**](cmdqueryname=service+deactivate-record)
  
  
  
  The EDSG service deactivation record function is enabled.
